Event Programming
模拟的基本单位Event
	send
	receive
	etc.
Event manager: Heap 每次取出最早的event处理
Event Handler: 模拟一个event
	单个event：分为sender event 和receiver event
	sender
		a traffic module to generate frames to be sent to the receiver;
		key elements of an ARQ protocol: ACKs (and NAKs), sequence number, timeout, send window 
		A queueing or buffering module to handle multiple transmissions and retransmissions 
	receiver
		a module to check if a received frame is accepted or rejected and calculate performance results 
		key elements of an ARQ protocol: replying ACKs or NAKs (indicating expected frame NO.), sequence number, receive window 
		A queueing or buffering module to handle out of sequence reception
模拟的过程：
	生成第一个event （frame generation）
	while （）do
		处理堆顶event

event 类别：
1. frame generation
	生成很多个frame，让sender发送（生成很多个send frame event）
2. send frame
	计算Tx
	计算delay
	生成一个receive frame event
3. receive frame
	计算frame error，随机模拟
	如果没有error：生成一个sender receive ACK event
	否则生成一个sender receive NAK event
4. sender receive ACK event
5. sender receive NAK event
